Python-API Project

"What's the weather like as we approach the equator?"
Now, we know what you may be thinking: "Duh. It gets hotter..."
But, if pressed, how would you prove it?

Create a Python script to visualize a random sampling of 500 cities across the world and show the relationship between Temperature versus Latitude, Humidity, Cloudiness, and Windspeed.Seperate the cities be Southern and Northern Hemisphere and run the same analysis.

Create a heatmap showing humidity levels for each city. Then narrow down the parameters to find a few cities that meet the following parameters:

  • max temp 80 degrees
  • min temp 70 degrees
  • wind speed less than 10 mph
  • zero cloudiness
